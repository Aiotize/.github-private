Aiotize Brand Guidelines

Welcome to the Aiotize brand guidelines. This document demonstrates our custom layout, branding, spacing, margins, typography hierarchy, and theme capabilities. Our design system supports light, dark, and custom themes with consistent styling across all platforms.

This page serves as a living example of our brand standards, including color palettes, fonts, hyperlink styles, numbering conventions, and imagery parameters. Each element has been carefully crafted to ensure brand consistency and optimal user experience.

Spacing & Layout

Spacing Scale

Our spacing system uses a consistent scale based on a 0.25rem (4px) unit. This ensures visual harmony and alignment throughout the design.

  • spacing-1 = 0.25rem (4px)
  • spacing-2 = 0.5rem (8px)
  • spacing-4 = 1rem (16px)
  • spacing-8 = 2rem (32px)
  • spacing-16 = 4rem (64px)

Margins & Padding

Page margins: 3rem vertical, 2rem horizontal
Section margins: 2rem vertical
Card padding: 1.5rem
Button padding: 0.5rem vertical, 1rem horizontal

Containers

We provide multiple container sizes for responsive layouts:

  • Small: 640px - For narrow content
  • Medium: 768px - For standard content
  • Large: 1024px - For wide content
  • Extra Large: 1280px - For full-width layouts
  • 2XL: 1536px - For ultra-wide displays

Hyperlinks & Navigation

Link Styles

Our hyperlinks are styled for clarity and accessibility. They feature:

  • Default link state - Primary color with subtle underline
  • Hover state - Secondary color with enhanced underline
  • Visited state - Purple to indicate visited links
  • Active state - Accent color when clicked

All links have proper focus states for keyboard navigation and meet WCAG accessibility standards.

Imagery & Graphics Guidelines

Image Parameters

Our imagery guidelines ensure consistent visual communication across all brand materials. We define specific parameters for lens, tone, and warmth to maintain brand identity.

Standard Lens Parameters

Focal Length: 50mm equivalent (natural perspective)

Aperture: f/5.6 (balanced depth of field)

Perspective: Natural, minimal distortion

Usage: Product photography, portraits, general use

Tone & Mood

Professional Tone: Medium contrast, 85% saturation, neutral brightness

Friendly Tone: Low contrast, 95% saturation, bright

Dramatic Tone: High contrast, 75% saturation, varied brightness

Warmth Settings

Cool (6500K): Blue tint - Technology, modern applications

Neutral (5500K): Balanced - General, versatile use

Warm (3200K): Orange tint - Comfort, trust, welcoming
